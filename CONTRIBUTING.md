# Contributing

Cold Review should become more useful, not merely harsher.

Good contributions:

- reduce unsupported certainty;
- improve evidence and citation requirements;
- make competitor comparisons more defensible;
- improve the distinction between uniqueness and different wording;
- produce cheaper, more falsifiable tests;
- shorten the output without removing important reasoning.

Please avoid:

- theatrical cruelty or insults;
- generic startup advice;
- invented market data;
- false precision;
- treating every missing feature as a market opportunity;
- encouraging users to copy competitors rather than understand them.

Open an issue before proposing a large structural change. For focused
improvements, submit a pull request explaining the failure mode the change
addresses.
