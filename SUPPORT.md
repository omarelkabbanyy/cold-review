# Support

## Usage Instructions

To use Cold Review, copy the contents of `PROMPT.min.txt` and paste it into an AI with web browsing capabilities. Then, fill out `INPUT_TEMPLATE.md` with your project details and submit it to the AI.

## Reporting a Bad Output

If you receive a weak, misleading, or unsupported output:
1. Check the issue tracker for similar problems.
2. Open a "Prompt problem" issue using the provided issue template.
3. Include the prompt version, the model used, the sanitized project input, and a description of the failure.

## Proposing Improvements

1. Ensure your improvement aligns with the [Design Principle](README.md#design-principle).
2. Open an "Improvement proposal" issue.
3. Discuss the change, its evidence basis, and tradeoffs.

## Limitations

Maintainers control the prompt's instructions but have no control over third-party AI models. We cannot fix fundamental model limitations, hallucinations, or browsing failures caused by external providers.
