# Project Input Template

Copy this underneath the Cold Review prompt. Short, concrete answers are better
than polished marketing language. If you don't know the answer, write `unknown` or `none` rather than inventing information.

```text
Project:

Project type:
[Business / App / Product / Service / Content / Other]

Current stage:
[Idea / Prototype / Launched / Revenue / Scaling]

Target user:

Geographic market:

Problem:

Current behavior or workaround:

How the project works:

Why someone would switch:

Known alternatives or competitors:

Pricing or business model:

Current distribution or acquisition method:

Evidence:

Evidence dates and sample sizes:

Usage and retention:

Feedback:

Previous tests and failures:

Constraints:

Legal, safety, privacy, security, or regulatory constraints:

Decision I need to make:
```
